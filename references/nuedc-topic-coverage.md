# NUEDC Topic Coverage

本清单覆盖当前收集到的历年资料题名，用于让其他 agent 按“年份/期次 + 题名”检索题面。它不包含本地路径、题面全文或个人信息。

适用性：`直接适用` 表示视觉为题目核心或明确可复用；`视觉辅助` 表示必须回读题面确认，不能默认使用摄像头；`非视觉不适用` 表示不应强行给出视觉方案。遇到陌生题，先检索相似模式；无适配时按题面重新拆解并询问用户。
